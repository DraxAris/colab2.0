Installing and setup of Colab
How to install and setup Colab?
  1. Put the Colab folders in the htcdocs of your hosting program
  2. Go to the server adres example: localhost/colab/pages
  3. Follow the instructions on screen
To use Colab you will need a program like Mamp to host Colab.

Adding Modules in Colab
What are Modules?

Modules are parts you can add to Colab to let Colab integrate with other services or even to use and create your own services for Colab that you and your organistion can use. Modules for Colab can codded in: Php, Javascript, Html, Css, Jquery, Python, ... You can use modules to change the style of the page or add new functions to Colab.

How do I add Modules?

  1. Go to the Modules folder.
  2. Create a new folder for your module
  3. Link your module on the page in the folder pages or head (if you want to add css)
  4. Restart the hosting programs
We advice you to add Modules when nobody is using the website or copy the Colab files to add it their first and then upload it to the server after testing it out.

Adding Guest Accounts
For what should you use it?

With guests account you can give visitors, parents, ... acces to a limited part of the system during their visit. You can block sertain parts of Colab for them you can give them acces to certain files, ...

How do I add a guest account?

  1. Login as an Admin
  2. Go to Add
  3. Click add user
  4. Select as user function guest
  5. Enter a username and password (password is not needed for a guest account)
  6. Click add
Guest accounts normaly only exist for 48 hours! If you want them to be able to acces their account longer then click on the time button. A guest account will be removed automaticly after 5 days!
To add a normal user you just follow the same steps but you select as a user function: normal user.

All user functions:

  • Super Admin (first user created)
  • Admin
  • Moderator
  • Normal user
  • Guest
  • Parent
  • Teacher
  • Director
